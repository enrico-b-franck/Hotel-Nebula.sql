# Hotel-Nébula


Projeto de modelagem e implementação de banco de dados para gerenciamento de um sistema hoteleiro.

O objetivo do projeto é estruturar os dados da rede fictícia **Hotel Nebula**, permitindo organizar reservas, hospedagens, pagamentos, avaliações, serviços extras e gestão operacional do hotel.

---

# 📌 Objetivos do Projeto

O projeto foi dividido em três etapas principais:

## Parte 1 — Origem do Sistema
- Levantamento de requisitos
- Identificação das entidades
- Regras de negócio
- Modelagem conceitual
- Criação do Diagrama ER

## Parte 2 — Montagem do Núcleo
- Criação do banco de dados relacional
- Implementação das tabelas
- Definição de chaves primárias e estrangeiras
- Inserção de dados iniciais

## Parte 3 — Radar de Comando
- Desenvolvimento de consultas SQL
- Extração de informações relevantes para gestão
- Análise operacional do hotel

---

# 🧩 Entidades do Sistema

O banco de dados contempla as seguintes entidades:

- Hóspedes
- Quartos
- Reservas
- Hospedagens
- Pagamentos
- Funcionários
- Serviços Extras
- Avaliações
